1. (1) 当 $a=0$ 时，$x\neq0$；当 $a>0$ 时，$x<-4a$ 或 $x>2a$；当 $a<0$ 时，$x<2a$ 或 $x>-4a$。
   (2) $a<x<b$。
   (3) 当 $a=0$ 或 $a=1$ 时，$x\neq -a$；当 $a<0$ 或 $a>1$ 时，$x<-a$ 或 $x>-a^2$；当 $0<a<1$ 时，$x<-a^2$ 或 $x>-a$。
   (4) 当 $|a|<4$ 时，$\mathbb{R}$；当 $|a|=4$ 时，$x\neq -\frac{a}{4}$；当 $|a|>4$ 时，$x<\frac{-a-\sqrt{a^2-16}}{4}$ 或 $x>\frac{-a+\sqrt{a^2-16}}{4}$。
   (5) 当 $|a|\le4$ 时，无解；当 $a<-4$ 或 $a>4$ 时，$\frac{a-\sqrt{a^2-16}}{2}<x<\frac{a+\sqrt{a^2-16}}{2}$。
   (6) 当 $a=1$ 时，$x\ge -\frac{1}{3}$；当 $a>1$ 时，$x\le \frac{-3-\sqrt{9-4(a-1)}}{2(a-1)}$ 或 $x\ge \frac{-3+\sqrt{9-4(a-1)}}{2(a-1)}$；当 $a<1$ 且 $\Delta=9-4(a-1)\ge0$ 即 $a\le \frac{13}{4}$ 时，$\frac{-3+\sqrt{9-4(a-1)}}{2(a-1)}\le x\le \frac{-3-\sqrt{9-4(a-1)}}{2(a-1)}$；当 $a<1$ 且 $\Delta<0$ 即 $a>\frac{13}{4}$ 时，无解。

2. 由解集得 $a=5, b=-6$，代入得 $-6x^2-5x-1>0$，即 $6x^2+5x+1<0$，解集为 $-\frac{1}{2}<x<-\frac{1}{3}$。

3. (1) 移项通分得 $\frac{2(x+a)-(4x-5)}{2(4x-5)}\le0$，即 $\frac{-2x+2a+5}{4x-5}\le0$，等价于 $(2x-2a-5)(4x-5)\ge0$ 且 $x\neq\frac{5}{4}$。当 $a=-\frac{5}{4}$ 时，$x<\frac{5}{4}$；当 $a>-\frac{5}{4}$ 时，$x\le\frac{5}{4}$ 或 $x\ge a+\frac{5}{2}$；当 $a<-\frac{5}{4}$ 时，$x\le a+\frac{5}{2}$ 或 $x>\frac{5}{4}$。
   (2) $\frac{3x+1}{2x-a}>-1$ 化为 $\frac{3x+1+2x-a}{2x-a}>0$，即 $\frac{5x+1-a}{2x-a}>0$，等价于 $(5x+1-a)(2x-a)>0$。当 $a=-\frac{1}{3}$ 时，$x\neq -\frac{1}{6}$；当 $a<-\frac{1}{3}$ 时，$x<\frac{a}{2}$ 或 $x>\frac{a-1}{5}$；当 $a>-\frac{1}{3}$ 时，$x<\frac{a-1}{5}$ 或 $x>\frac{a}{2}$。
   (3) $\frac{2x-m}{5x-3}>-2$ 化为 $\frac{2x-m+2(5x-3)}{5x-3}>0$，即 $\frac{12x-m-6}{5x-3}>0$，等价于 $(12x-m-6)(5x-3)>0$。当 $m=-\frac{12}{5}$ 时，$x\neq\frac{3}{5}$；当 $m<-\frac{12}{5}$ 时，$x<\frac{m+6}{12}$ 或 $x>\frac{3}{5}$；当 $m>-\frac{12}{5}$ 时，$x<\frac{3}{5}$ 或 $x>\frac{m+6}{12}$。
   (4) $\frac{2-x}{4x-m}\le\frac{1}{2}$ 化为 $\frac{2(2-x)-(4x-m)}{2(4x-m)}\le0$，即 $\frac{-6x+4+m}{4x-m}\le0$，等价于 $(-6x+4+m)(4x-m)\le0$ 且 $x\neq\frac{m}{4}$。当 $m=\frac{16}{5}$ 时，$x<\frac{4}{5}$；当 $m<\frac{16}{5}$ 时，$x\le\frac{m}{4}$ 或 $x\ge\frac{4+m}{6}$；当 $m>\frac{16}{5}$ 时，$x\le\frac{4+m}{6}$ 或 $x>\frac{m}{4}$。
   (5) $\frac{ax-1}{2x+3}\ge0$ 等价于 $(ax-1)(2x+3)\ge0$ 且 $x\neq -\frac{3}{2}$。当 $a=0$ 时，$x<-\frac{3}{2}$；当 $a>0$ 时，$x\le -\frac{3}{2}$ 或 $x\ge\frac{1}{a}$；当 $a<0$ 时，$\frac{1}{a}\le x< -\frac{3}{2}$。
   (6) $\frac{mx-2}{4x+1}<1$ 化为 $\frac{mx-2-4x-1}{4x+1}<0$，即 $\frac{(m-4)x-3}{4x+1}<0$，等价于 $((m-4)x-3)(4x+1)<0$。当 $m=4$ 时，$x>-\frac{1}{4}$；当 $m>4$ 时，$-\frac{1}{4}<x<\frac{3}{m-4}$；当 $m<4$ 时，$x<-\frac{1}{4}$ 或 $x>\frac{3}{m-4}$。

4. 变式1：$\frac{mx+5}{2x-3}\le -1$ 化为 $\frac{mx+5+2x-3}{2x-3}\le0$，即 $\frac{(m+2)x+2}{2x-3}\le0$，等价于 $((m+2)x+2)(2x-3)\le0$ 且 $x\neq\frac{3}{2}$。当 $m=-2$ 时，$x<\frac{3}{2}$；当 $m>-2$ 时，$-\frac{2}{m+2}\le x<\frac{3}{2}$；当 $m<-2$ 时，$x\le -\frac{2}{m+2}$ 或 $x>\frac{3}{2}$。
   变式2：$\frac{ax-2}{3x+1}>-1$ 化为 $\frac{ax-2+3x+1}{3x+1}>0$，即 $\frac{(a+3)x-1}{3x+1}>0$，等价于 $((a+3)x-1)(3x+1)>0$。当 $a=-3$ 时，$x>-\frac{1}{3}$；当 $a>-3$ 时，$x<-\frac{1}{3}$ 或 $x>\frac{1}{a+3}$；当 $a<-3$ 时，$\frac{1}{a+3}<x<-\frac{1}{3}$。
