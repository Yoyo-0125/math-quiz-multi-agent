1. 当 $a > -\frac{1}{2}$ 时，解集为 $\{x \mid x < \frac{1}{2}\}$；当 $a = -\frac{1}{2}$ 时，解集为 $\{x \mid x < \frac{1}{2}\}$；当 $a < -\frac{1}{2}$ 时，解集为 $\{x \mid \frac{1}{2} < x \le -a\}$。
  简要步骤：移项通分得 $\frac{2a+1}{2(2x-1)} \le 0$，即 $\frac{2a+1}{2x-1} \le 0$，分子为常数 $2a+1$，分类讨论 $2a+1$ 的正负，同时注意分母 $2x-1 \neq 0$，且 $x \le -a$ 还需满足 $x \neq \frac{1}{2}$。
