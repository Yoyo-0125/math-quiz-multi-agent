1. 当 $a=0$ 时，解集为 $\{x\mid x\neq0\}$；当 $a>0$ 时，解集为 $\{x\mid x<-4a \text{ 或 } x>2a\}$；当 $a<0$ 时，解集为 $\{x\mid x<2a \text{ 或 } x>-4a\}$。

2. 解集为 $\{x\mid a<x<b\}$。

3. 当 $a=0$ 或 $a=1$ 时，解集为 $\{x\mid x\neq -a\}$；当 $a<0$ 或 $a>1$ 时，解集为 $\{x\mid x<-a\text{ 或 }x>-a^2\}$；当 $0<a<1$ 时，解集为 $\{x\mid x<-a^2\text{ 或 }x>-a\}$。

4. 当 $|a|\leq 4$ 时，解集为 $\mathbb{R}$；当 $a<-4$ 时，解集为 $\{x\mid x<\frac{-a-\sqrt{a^2-16}}{4}\text{ 或 }x>\frac{-a+\sqrt{a^2-16}}{4}\}$；当 $a>4$ 时，解集为 $\{x\mid x<\frac{-a-\sqrt{a^2-16}}{4}\text{ 或 }x>\frac{-a+\sqrt{a^2-16}}{4}\}$（注意 $a>4$ 时两根为负）。

5. 当 $-4<a<4$ 时，解集为空集；当 $a=4$ 或 $a=-4$ 时，解集为空集（$x=\pm2$ 处等于0）；当 $a<-4$ 时，解集为 $\{x\mid \frac{a-\sqrt{a^2-16}}{2}<x<\frac{a+\sqrt{a^2-16}}{2}\}$；当 $a>4$ 时，解集为 $\{x\mid \frac{a-\sqrt{a^2-16}}{2}<x<\frac{a+\sqrt{a^2-16}}{2}\}$。

6. 当 $a=1$ 时，解集为 $\{x\mid x\geq -\frac{1}{3}\}$；当 $a>1$ 时，$\Delta=13-4a$，若 $\Delta<0$ 即 $a>\frac{13}{4}$，解集为 $\mathbb{R}$；若 $\Delta=0$ 即 $a=\frac{13}{4}$，解集为 $\{x\mid x=-\frac{2}{3}\}$；若 $\Delta>0$ 即 $1<a<\frac{13}{4}$，解集为 $\{x\mid x\leq\frac{-3-\sqrt{13-4a}}{2(a-1)}\text{ 或 }x\geq\frac{-3+\sqrt{13-4a}}{2(a-1)}\}$；当 $a<1$ 时，$\Delta=13-4a$，若 $a<\frac{13}{4}$ 恒成立，$a<1$ 时 $\Delta>0$，解集为 $\{x\mid \frac{-3+\sqrt{13-4a}}{2(a-1)}\leq x\leq\frac{-3-\sqrt{13-4a}}{2(a-1)}\}$（注意 $a-1<0$ 不等号反向）。

7. 由根与系数关系得 $a=5,b=-6$，不等式 $bx^2-ax-1>0$ 即 $-6x^2-5x-1>0$，解得 $\{x\mid -\frac{1}{2}<x<-\frac{1}{3}\}$。

8. 化为 $\frac{-2x+2a+5}{4x-5}\leq 0$，即 $(2x-2a-5)(4x-5)\geq 0$ 且 $x\neq\frac{5}{4}$。当 $a=-\frac{5}{2}$ 时，解集为 $\{x\mid x<\frac{5}{4}\}$；当 $a>-\frac{5}{2}$ 时，解集为 $\{x\mid x\leq\frac{5}{4}\text{ 或 }x\geq a+\frac{5}{2}\}$；当 $a<-\frac{5}{2}$ 时，解集为 $\{x\mid x\leq a+\frac{5}{2}\text{ 或 }x>\frac{5}{4}\}$。

9. 化为 $\frac{5x+1-a}{2x-a}>0$，即 $(5x+1-a)(2x-a)>0$。当 $a=\frac{2}{7}$ 时，解集为 $\{x\mid x\neq\frac{1}{7}\}$；当 $a>\frac{2}{7}$ 时，解集为 $\{x\mid x<\frac{a}{2}\text{ 或 }x>\frac{a-1}{5}\}$；当 $a<\frac{2}{7}$ 时，解集为 $\{x\mid x<\frac{a-1}{5}\text{ 或 }x>\frac{a}{2}\}$。

10. 化为 $\frac{12x-6-m}{5x-3}>0$，即 $(12x-6-m)(5x-3)>0$。当 $m=-\frac{42}{5}$ 时，解集为 $\{x\mid x\neq\frac{3}{5}\}$；当 $m>-\frac{42}{5}$ 时，解集为 $\{x\mid x<\frac{3}{5}\text{ 或 }x>\frac{6+m}{12}\}$；当 $m<-\frac{42}{5}$ 时，解集为 $\{x\mid x<\frac{6+m}{12}\text{ 或 }x>\frac{3}{5}\}$。

11. 化为 $\frac{-6x+4+m}{4x-m}\leq 0$，即 $(6x-4-m)(4x-m)\geq 0$ 且 $x\neq\frac{m}{4}$。当 $m=-\frac{8}{5}$ 时，解集为 $\{x\mid x\neq -\frac{2}{5}\}$；当 $m>-\frac{8}{5}$ 时，解集为 $\{x\mid x\leq\frac{m}{4}\text{ 或 }x\geq\frac{4+m}{6}\}$；当 $m<-\frac{8}{5}$ 时，解集为 $\{x\mid x\leq\frac{4+m}{6}\text{ 或 }x>\frac{m}{4}\}$。

12. 等价于 $(ax-1)(2x+3)\geq 0$ 且 $x\neq -\frac{3}{2}$。当 $a=0$ 时，解集为 $\{x\mid x<-\frac{3}{2}\}$；当 $a<0$ 时，令 $\frac{1}{a}$ 与 $-\frac{3}{2}$ 比较：若 $a<0$ 且 $\frac{1}{a}>-\frac{3}{2}$ 即 $a<-\frac{2}{3}$，解集为 $\{x\mid \frac{1}{a}\leq x<-\frac{3}{2}\}$；若 $a=-\frac{2}{3}$，解集为 $\emptyset$；若 $-\frac{2}{3}<a<0$，解集为 $\{x\mid -\frac{3}{2}<x\leq\frac{1}{a}\}$；当 $a>0$ 时，$\frac{1}{a}>-\frac{3}{2}$ 恒成立，解集为 $\{x\mid x<-\frac{3}{2}\text{ 或 }x\geq\frac{1}{a}\}$。

13. 化为 $\frac{(m-4)x-3}{4x+1}<0$，即 $((m-4)x-3)(4x+1)<0$。当 $m=4$ 时，解集为 $\{x\mid x>-\frac{1}{4}\}$；当 $m>4$ 时，解集为 $\{x\mid -\frac{1}{4}<x<\frac{3}{m-4}\}$；当 $m<4$ 时，解集为 $\{x\mid x<-\frac{1}{4}\text{ 或 }x>\frac{3}{m-4}\}$。

14. 化为 $\frac{mx+5+2x-3}{2x-3}\leq 0$，即 $\frac{(m+2)x+2}{2x-3}\leq 0$，等价于 $((m+2)x+2)(2x-3)\leq 0$ 且 $x\neq\frac{3}{2}$。当 $m=-2$ 时，解集为 $\{x\mid x<\frac{3}{2}\}$；当 $m>-2$ 时，解集为 $\{x\mid -\frac{2}{m+2}\leq x<\frac{3}{2}\}$；当 $m<-2$ 时，解集为 $\{x\mid x<\frac{3}{2}\text{ 或 }x\geq -\frac{2}{m+2}\}$。

15. 化为 $\frac{ax-2+3x+1}{3x+1}>0$，即 $\frac{(a+3)x-1}{3x+1}>0$，等价于 $((a+3)x-1)(3x+1)>0$。当 $a=-3$ 时，解集为 $\{x\mid x<-\frac{1}{3}\}$；当 $a>-3$ 时，解集为 $\{x\mid x<-\frac{1}{3}\text{ 或 }x>\frac{1}{a+3}\}$；当 $a<-3$ 时，解集为 $\{x\mid \frac{1}{a+3}<x<-\frac{1}{3}\}$。
